Used Technologies

This project requires following tools and technologies:
  • Java: project was tested on version 1.8.0_221
  • Maven: project was tested on version 3.3.9
  • Spring Boot MVC: used Back-End technology. Project was tested on version 2.3.4.RELEASE
  • Thymeleaf: used Front-End technology. Project was tested on version 3.0.4.RELEASE
  • Bootstrap: used CSS framework. Project was tested on version 5
  • GIT (optional): this tool is only required for downloading source code. Project was tested on version 2.22.0

Prerequisites

This project is based on existing project mvc-thymeleaf. Article describes only differences between these two projects. Base project is available on my Github repository. You can:
  • Download base project mvc-thymeleaf directly from my Github Repository;
  • Or download base project using following commands:
$ git clone git@github.com:wisniewskikr/spring-boot.git
$ cd mvc-thymeleaf

Project Structure

This project has following structure of files:

Image: Projec structure

Step 1. Update command class GreetingCommand.java

The class GreetingCommand.java is an object whose role is to store all the information required for executing an action.
Important lines:
  • line 5: field name type String;
  • line 6: field message type String.

Step 2. Update controller class GreetingController.java

The class GreetingController.java controls the flow of the data. It controls the data flow into model object and updates the view whenever data changes.
Important lines:
  • lines 14 - 17: method display() displays page Greeting;
  • lines 19 - 26: method handleSubmit() handles submit form page Greeting - takes value of name and put it into message. Then again displays page Greeting.

Step 3. Update HTML file greeting.html

The file greeting.html contains all elements which are designed to be displayed in a web browser.
Important lines:
  • lines 20 - 30: form in thymeleaf. This form contains input name and div message. Values from form are sent to location described as action and with method described as method after click for button type submit.

Step 4. Update CSS file custom.css

The file custom.css describes the presentation of HTML elements.
Important lines:
  • lines 1 - 7: styles definition.

Run Project

You can run this project:
  • Directly launching class Application.java in your IDE;
  • Or using following Maven commands:
$ mvn spring-boot:run

To see demo please copy and paste following URL to your browser's address field: http://localhost:8080/app/greeting.
